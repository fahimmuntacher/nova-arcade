NovaArcade is a modern gaming platform where users can explore, play, and discover their favorite games. Built with **React**, **Tailwind CSS**, **Framer Motion**, and **Firebase**, it offers smooth animations, secure authentication, and an engaging UI.

---

## Features

- User authentication with **Firebase** (email/password & Google OAuth)
- Email verification for secure login
- Forgot password functionality
- Explore games & developers
- Responsive UI with **Tailwind CSS**
- Smooth animations with **Framer Motion**
- Newsletter subscription section
- Custom footer with social links
- Protected routes to restrict access for unauthorized users
